- 架构: 栈式虚拟机
- 数据类型: 64位整数 (可扩展支持浮点或对象)
- 内存模型:
  - 操作数栈 (最大 1024 个元素)
  - 全局内存 (最大 4096 个槽位, 用于变量存储)
  - 代码段 (字节码数组)
- 指令集 (单字节操作码 + 可选立即数):

| 操作码 | 名称    | 说明                                       |
|--------|---------|--------------------------------------------|
| 0x01   | PUSH    | 推入一个立即数 (8字节) 到栈顶              |
| 0x02   | POP     | 弹出栈顶值, 丢弃                           |
| 0x03   | ADD     | 弹出a,b, 推入a+b                           |
| 0x04   | SUB     | 弹出a,b, 推入b-a                           |
| 0x05   | MUL     | 弹出a,b, 推入a*b                           |
| 0x06   | DIV     | 弹出a,b, 推入b/a (整数除法)                |
| 0x07   | STORE   | 弹出值, 存入全局内存[addr] (地址为1字节)   |
| 0x08   | LOAD    | 从全局内存[addr]加载值入栈                 |
| 0x09   | JMP     | 无条件跳转到目标地址 (2字节偏移)           |
| 0x0A   | JMP_IF  | 弹出条件, 非零则跳转 (2字节偏移)           |
| 0x0B   | HALT    | 停止虚拟机执行                             |
| 0x0C   | PRINT   | 弹出值并打印 (调试用)                      |
| 0x0D   | DUP     | 复制栈顶元素                               |
| 0x0E   | SWAP    | 交换栈顶两个元素                           |

- 错误处理: 返回错误码, 通过导出函数获取错误信息。
